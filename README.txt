# 圈X (Quantumult X) 完整配置

一个文件，一次粘贴，全部搞定。

## 使用方法

**1.** 打开 **圈X → ⚙️ → 配置文件 → 编辑**

**2.** 全选原有内容 → 全部删除

**3.** 打开 `quanx_full.conf` 文件 → 全选复制 → 粘贴到圈X配置编辑器

**4.** 右上角 **保存**，等待圈X自动下载远程规则（约10-30秒），即可生效

> 远程规则来自 blackmatrix7 维护的规则库，每24小时自动更新，无需手动维护。
